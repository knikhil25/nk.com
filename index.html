What I've Built

Projects that got out of hand (in the best way).

🤖
Rubik's Cube Solving Robot
A robot that reads a scrambled cube with a webcam using OpenCV and then physically solves it using 28BYJ-48 stepper motors driven by an Arduino Mega. Frame designed in CAD and 3D printed in PLA+.
Arduino Mega OpenCV Python 3D Printing Stepper Motors
🌿
Dracaena Bot / Basil Bot
Science fair project giving a plant "emotions." Uses a capacitive touch sensor and soil moisture sensor to detect interactions, displaying feelings on an LCD. The plant literally tells you when it's thirsty.
Arduino LCD Display Moisture Sensor Capacitive Touch
🚚
Home Delivery Robot
Indoor navigation robot that finds its way around the house using ArUco marker detection on walls. ESP32-CAM handles vision + WiFi; Arduino Mega controls the motors. Budget: ~$153.
ESP32-CAM ArUco Markers Arduino Mega Navigation
🧩
Cube Scramble Trainer
Browser-based color input UI for solving scrambles — built after OpenCV kept confusing red and orange stickers. Full solve visualization with algorithm output.
HTML/CSS/JS Kociemba Algorithm WebGL
📅
Smart Schedule System
Built a structured daily timetable with school, homework, Math Olympiad, drums, and downtime blocks — exported to Google Sheets. Work smarter, not harder.
Google Sheets Time Management Automation
